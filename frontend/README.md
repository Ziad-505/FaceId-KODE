# KODE Face-ID — Frontend

React + Vite single-page application for the Face-ID enrollment portal.

See the **[root README](../README.md)** for setup and **[docs/PORTAL.md](../docs/PORTAL.md)** for architecture.

## Quick commands

```bash
cp .env.example .env
npm install
npm run dev      # http://localhost:3000
npm run build    # output → dist/
```

Place your portal logo at `public/logo.png`.
