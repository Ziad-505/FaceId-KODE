# KODE Face-ID — Backend

Express + PostgreSQL API for portal authentication, RBAC, audit logging, and dashboard aggregation.

See the **[root README](../README.md)** for setup and the **[architecture guide](../docs/PORTAL.md)** for layer details.

## Quick commands

```bash
cp .env.example .env
npm install
npm run migrate
npm run seed
npm run dev      # http://localhost:4000
npm test
```
